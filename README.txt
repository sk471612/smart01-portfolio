# 학생용 포트폴리오 템플릿 세트

## 구성 파일

- index.html : 메인 페이지
- about.html : 자기소개 페이지
- activity.html : 학습 및 활동이력 페이지
- projects.html : 프로젝트 목록 페이지
- contact.html : 연락처 페이지
- projects/project1.html : 프로젝트 상세 페이지 1
- projects/project2.html : 프로젝트 상세 페이지 2
- projects/project3.html : 프로젝트 상세 페이지 3
- css/style.css : 전체 디자인 스타일

## 사용 방법

1. 압축을 풉니다.
2. 학생 이름, 학년, 관심 분야, 프로젝트 내용을 수정합니다.
3. GitHub 저장소에 전체 파일을 업로드합니다.
4. Settings > Pages에서 Deploy from a branch를 선택합니다.
5. Branch를 main, folder를 /root로 설정합니다.

## 주의사항

- 메인 파일명은 반드시 index.html이어야 합니다.
- css 폴더와 projects 폴더 구조를 바꾸면 링크가 깨질 수 있습니다.
- 이미지 주소는 원하는 사진 주소 또는 images 폴더의 파일명으로 변경할 수 있습니다.
